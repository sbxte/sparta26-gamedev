## Deskripsi Project
Project ini merupakan game 2D side-scrolling runner yang menceritakan perjuangan Special Week dalam menyelesaikan latman 'WI2003 Olah Raga' di ITB Kampus Jatinangor.

## Cara Menjalankan Game
Run executable file yang telah di export dari Godot. 

## Cara Melakukan Build
Clone repository:
```bash
git clone https://github.com/sbxte/sparta26-gamedev
```
Lalu buka repository di godot dan export untuk mendapat file executable, atau run di godot langsung.

## Daftar Control
W/Space - Jump <br>
S - Slide <br>
Shift - Boost <br>

## Daftar Fitur
Jump: Lompat untuk menghindari obstacle di jalan <br>
Slide: Meluncur untuk menghindati obstacle yang terbang <br>
Boost: Meningkatkan kecepatan dan menghancurkan obstacle <br>
Difficulty Selection: Di setiap session, player dapat memilih salah satu dari 3 tingkat kesulitan yang akan berdampak pada penempatan obstacle pada session <br>
